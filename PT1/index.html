Breve Histórico

Um requesito é uma característica do sistema ou a descrição de algo que o sistema pe capaz de realizar para atingir seus obejetivos, sendo assim a Gerência de Requesitos (GRE) é um congunto de atividades que ajudam a equipe de projeto a identificat, controlar e restrear requesitos e modificações de requisitos em qualquer época, à medida que o projeto prossegue.
